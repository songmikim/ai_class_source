자료형 
- 기본 자료형
	- 숫자를 저장하는 자료형 

- 참조 자료형
	- 다른 데이터의 주소를 참조 하는 자료형 
	- 값으로 주소를 가지는 자료형
	
2. 기본 자료형의 종류 *****
    : 숫자를 저장하기 위한 목적의 자료형
    1) 정수형 자료형 
        byte : 1바이트, -2^7~2^7 - 1 (-128~127) 
        short: 2바이트 -2^15~2^15 - 1 
        int  : 4바이트 -2^31 ~ 2^31 - 1
        long : 8바이트 -2^63 ~ 2^63 - 1

    2) 실수형 자료형 - 소수점까지 표현하는 자료형
        float : 4바이트
        double : 8바이트

    3) 문자 자료형
        char (2~3바이트)
            : 값은 문자 1개, '문자'
            : 양의 정수 (unsigned)
    4) 논리 자료형
        boolean : 1바이트 (true(참), false(거짓))
