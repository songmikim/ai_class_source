DCL (Data Control Language) 
	- 권한, 데이터의 저장시 통제
	권한 : GRANT, REVOKE 
	통제 : COMMIT, ROLLBACK -> TCL(Transaction Control Language) 
	
	Transaction : 트랜잭션 - SQL 실행 단위 (COMMIT 할때 까지) 
	COMMIT - 데이터베이스에 영구 반영
